# 🧬 FitGenie

> AI-powered personalized nutrition and workout planner.

FitGenie is an intelligent fitness application that generates personalized meal plans and workout routines based on each user's goals, lifestyle, and preferences. It leverages Google's Gemini AI to provide smart recommendations and nutritional insights, making fitness planning simple, personalized, and efficient.

---

## 🚀 Features

- Personalized user profiling based on fitness goals and lifestyle
- AI-generated full-day meal plans
- Customized 7-day workout routines
- Workout demonstration videos for every exercise
- Food image scanner for meal analysis
- Automatic nutritional breakdown
  - Calories
  - Protein
  - Carbohydrates
  - Fats
- AI-powered health and fitness recommendations
- Clean and intuitive user interface

---

## 🧠 Core Idea

FitGenie combines Artificial Intelligence with personalized fitness planning to generate customized nutrition and workout schedules for every user.

Instead of following generic diet charts or workout plans, users receive recommendations tailored specifically to their body goals, lifestyle, and dietary preferences.
